# Compound Interest Calculator

## Project Description

This project is a simple web-based Compound Interest Calculator built using HTML, CSS, and JavaScript.
It allows users to enter principal amount, rate of interest, compounding frequency, and time, and then calculates the compound interest.

## How to Run

1. Download or copy all project files:

   * `index.html`
   * `style.css`
   * `script.js`

2. Make sure all files are in the same folder.

3. Open the `index.html` file in any web browser.

4. Enter the required values:

   * Principal (P)
   * Rate of Interest (r)
   * Compounding Frequency (n)
   * Time (t)

5. Click the **Calculate** button.

6. The compound interest will be displayed on the screen.

## Formula Used

The formula for compound interest is:

A = P(1 + r/n)^(nt)

Where:

* **A** = Final amount
* **P** = Principal amount
* **r** = Rate of interest (in decimal)
* **n** = Number of times interest is compounded per year
* **t** = Time (in years)

Compound Interest (CI) is calculated as:

CI = A - P

## Example Usage

Input:

* Principal (P) = 10000
* Rate (r) = 5%
* Frequency (n) = 1
* Time (t) = 3 years

Output:
Compound Interest ≈ 1576.25
